Congratulations, you've extracted a field! You could have selected multiple terms and extractions in the previous workflow, but fortunately Splunk already understands "access_combined" log formats and has fields available for us.

Below you'll see what's known as the Search view. You should familiarize yourself with this view. It's generally located by the "Search" link within the navigation menu toward the top left of this page, but has been placed inline for your convenience. This view is where most of your "ad hoc" searching, or search development will take place.

A search has been issued for you. If you're following along with different index names you'll need to change as appropriate.

Familiarize yourself with this view. The major components are:

  • Search bar and Timepicker The place to input search terms and select your search timeframe. You should always try to be as specific as possible with respect to both of these factors.
  • Events, Patterns, Statistics, and Visualization tabs Each tab will change the data view. The Events tab shows your raw search results. The Patterns tab shows the command patterns of the events. Statistics and Visualization will show tabular or graphical representation of the data when using a proper search command.
  • Timeline A visual representation of the count of results at the time of the events.
  • Fields Selected and most common field names are shown along the left side of the screen. Review the fields listed and click on a few.
  • Events The actual events that were sent to Splunk. You may click the ">" to the left of an event to see the extracted fields specific to it.
